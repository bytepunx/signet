== Quick-start ==

1. Expose the admin port via port-forward:

   kubectl port-forward -n {{ .Release.Namespace }} \
     deployment/{{ include "signet.fullname" . }} 8444:8444

2. Get a short-lived admin token:

   kubectl create token {{ include "signet.adminServiceAccountName" . }} \
     -n {{ .Release.Namespace }} --duration=1h

3. Check seal state:

   signet --server localhost:8444 --token <token-from-step-2> status

4. Unseal the server (direct-key mode):

   signet --server localhost:8444 --token <token> \
     unseal key --key-file /path/to/master.key

   Or in Shamir mode, collect threshold shares from keyholders:

   signet --server localhost:8444 --token <token> \
     unseal share --share <hex-share>
